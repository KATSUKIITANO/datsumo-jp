# 脱毛アフィリエイト KPI管理・撤退ルール

## 撤退・改善の判断基準（厳守）

| 指標 | 基準 | アクション |
|------|------|-----------|
| クリック数 300 / CV 0件 | 案件停止ライン | **即時停止・案件変更** |

## A/Bテスト管理

| | LP-A（index.html） | LP-B（lp-b.html） |
|---|---|---|
| タイプ | 比較ランキング型 | 悩み共感型ストーリー |
| メインキャッチ | 「30代からの脱毛おすすめ比較ランキング」 | 「毎朝の自己処理、もう疲れませんでしたか？」 |
| 想定ユーザー | 比較検討段階・情報収集中 | 悩み強め・感情的動機 |
| ターゲットキーワード | 「脱毛 おすすめ 30代」「医療脱毛 比較」 | 「自己処理 面倒」「脱毛 始めたい」「ムダ毛 30代」 |

### A/Bテスト判定基準
- 最低サンプル：各LP 500クリック以上
- 判定指標：CVR（コンバージョン率）
- 採用基準：CVRが高い方を本番採用
- テスト期間目安：1〜2週間
